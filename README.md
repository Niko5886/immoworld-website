# ImmoWorld - Real Estate Website

A modern, responsive real estate website built with HTML, CSS, and JavaScript.

## Features

- Responsive design
- Property listings
- Location-based search (Berlin, Hamburg, Frankfurt)
- Contact forms
- Blog section
- Modern UI with FontAwesome icons

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- Slippery.js (for sliders)
- FontAwesome 6.6.0

## Pages

- Homepage (`index.html`)
- About (`about.html`)
- Properties (`angebot1.html`, `angebot2.html`, `angebot3.html`)
- Cities (`Berlin.html`, `hamburg.html`, `frankfurt.html`)
- Blog (`blog.html`)
- Contact (`kontakt.html`)
- Sell (`verkaufen.html`)

## Setup

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - it's a static site

## Deployment

This site is deployed on Netlify and automatically updates when changes are pushed to the main branch.
